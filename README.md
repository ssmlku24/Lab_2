# Розв'язання прямої та двоїстої задач лінійного програмування

Цей консольний додаток мовою C# призначений для покрокового розв'язання задач лінійного програмування (ЗЛП) та їх двоїстих задач за допомогою методу жорданових виключень (МЖВ). Програма автоматично будує двоїсту задачу на основі введеної прямої та виводить детальний протокол обчислень.

## Основні можливості
* Розв'язання задач як на максимізацію (max), так і на мінімізацію (min).
* Підтримка змішаних систем обмежень (<=, >=, =).
* Покрокове виведення симплекс-таблиць на кожній ітерації алгоритму.
* Автоматичне видалення нуль-рядків (штучних змінних) за необхідності.
* Одночасне знаходження векторів розв'язків X (для прямої задачі), а також U та V (для двоїстої задачі).
* Експорт детального протоколу розв'язання у текстовий файл `.txt` для подальшого аналізу.

## Інструкція з використання
1. Запустіть скомпільований файл програми.
2. Введіть кількість змінних (n) та кількість обмежень (m).
3. Оберіть напрямок цільової функції (1 для max, 2 для min).
4. Введіть коефіцієнти цільової функції через пробіл.
5. Для кожного обмеження введіть:
   - Коефіцієнти при змінних (через пробіл).
   - Знак обмеження (<=, >= або =).
   - Вільний член (b).
6. Слідкуйте за покроковим розв'язанням у консолі. 
7. Після завершення обчислень програма запропонує зберегти результати у файл. Введіть `т` або `так` для підтвердження.

## Структура коду
* `DualExpression` — допоміжний клас для збереження виразів віддалених двоїстих змінних.
* `Main` — точка входу, ініціалізація меню.
* `SolveLPPMenu` — основний метод, який керує логікою введення даних, перетворенням таблиць та пошуком опорного й оптимального розв'язків.
* `DoSimplexMJEStep` — метод виконання одного кроку жорданових виключень.
